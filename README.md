# InAmigos Foundation - Web Development Internship

## Task 1: NGO Awareness Webpage

This repository contains the codebase for **Task 1** of my Web Development Internship at **InAmigos Foundation**. 

### 📋 Project Objective
The goal of this task was to design and develop a structured, clean, and responsive awareness webpage for the NGO using basic HTML and CSS based on their official branding guidelines and core initiatives.

### ✨ Features Included
* **Split Hero Section:** Features a concise, professional background introduction of the foundation alongside real campaign visuals.
* **Our Projects:** Clean grid cards detailing the 6 core initiatives (*Project Seva, Bachpanshala, Jeev, Udaan, Prakriti, and Vikas*).
* **Our Impact Metrics:** Sleek visual counter cards showcasing the scale of volunteers, states reached, and beneficiaries touched by the foundation.
* **Visual Gallery:** Custom-cropped responsive gallery grid showing ground-level campaign highlights safely framed without image distortion.
* **Centered Navigation & Dynamic Footer:** Polished sticky header layout with an organized 3-column metadata layout at the base.

### 🛠️ Tech Stack Used
* HTML5 (Semantic Structure)
* CSS3 (Flexbox, Grid Alignment, Custom Properties)
